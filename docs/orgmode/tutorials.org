* Bounds and FPair

The =FPair= and =NumLimit= primitives exposed today match the legacy
API surface; the typed Rust replacements land with v0.3.0 alongside
the =anneal= component algebra.

#+BEGIN_SRC python
import numpy as np
from eindir.core.components import NumLimit
b = NumLimit(low=np.array([-1., -1.]), high=np.array([1., 1.]), dims=2)
print(b.mkpoint())
#+END_SRC
